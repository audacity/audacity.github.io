<Callout type="info">
This feature is only available if you have **colorful clips** enabled in **Preferences -> Appearance -> Clip style**
</Callout>

## Changing color per clip

To change clip color, click on the 3-dot menu on the top right of the clip and select **Clip color**. You then are given the choice between several colors.

![](https://support.audacityteam.org/~gitbook/image?url=https%3A%2F%2F2905281570-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FEnNXGrLkN0pXlkzzS7cM%252Fuploads%252FWwCB9swh69hD8Z9d4WaP%252Fimage.png%3Falt%3Dmedia%26token%3D8c56a16c-655d-4393-aeb5-da343eef7e88&width=768&dpr=3&quality=100&sign=bc996ebf&sv=2)

## Changing color for an entire track

To change the color for all clips on a track, click on the 3-dot menu on the track header and select **Track color**. You then are given the choice between several colors.

![](https://support.audacityteam.org/~gitbook/image?url=https%3A%2F%2F2905281570-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FEnNXGrLkN0pXlkzzS7cM%252Fuploads%252FQdIaeTucdSL5MZBGCeTp%252Fimage.png%3Falt%3Dmedia%26token%3D5cbef214-2623-4bfc-9c3f-d4f685db615a&width=768&dpr=3&quality=100&sign=1d951a9c&sv=2)

<Callout type="info">
Changing track color only changes the color for clips which follow the track color. If you have changed the clip color for individual clips, they won't change color if you change track color.
</Callout>
