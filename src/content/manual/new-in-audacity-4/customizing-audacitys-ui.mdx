## Workspaces

### Turning on and off features

You can turn on and off individual features in Audacity's toolbar using the cog icon on the right. Doing so will show or hide certain UI elements in the toolbar.

![](https://support.audacityteam.org/~gitbook/image?url=https%3A%2F%2F2905281570-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FEnNXGrLkN0pXlkzzS7cM%252Fuploads%252F9BB833Cx5yK0zFyaYc1Q%252Fimage.png%3Falt%3Dmedia%26token%3D5b38599f-9a02-41af-bbe1-aab738878c96&width=768&dpr=3&quality=100&sign=3791b4ec&sv=2)

### Moving, docking and floating parts of the UI

Toolbars and panels can be moved by grabbing and dragging them at the top (like an app window), or the ⠿ 6-dot grabber of the element.

### Workspace presets

You can switch between different presets using the Workspace dropdown in the top right corner of Audacity.

![](https://support.audacityteam.org/~gitbook/image?url=https%3A%2F%2F2905281570-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FEnNXGrLkN0pXlkzzS7cM%252Fuploads%252FyB2zoWQAVG2O8yrPaTYg%252Fimage.png%3Falt%3Dmedia%26token%3Df903d7d3-126c-4d17-8ce3-67fac7111a9b&width=768&dpr=3&quality=100&sign=ac92667c&sv=2)

<Callout type="info">
All UI modifications you make are automatically saved to the workspace you're currently in.
</Callout>

## Themes

Audacity's theme, accent color and clip style can be changed in **Preferences -> Appearance**.

![](https://support.audacityteam.org/~gitbook/image?url=https%3A%2F%2F2905281570-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FEnNXGrLkN0pXlkzzS7cM%252Fuploads%252FPd6JzEMPGV3myzM3yj3H%252Fimage.png%3Falt%3Dmedia%26token%3D3b44e74d-55d6-4457-bac0-5bd17dc39131&width=768&dpr=3&quality=100&sign=3befea3&sv=2)

### Custom themes

It is possible to further customize Audacity's look using the config files located in the Audacity install folder's `/bin/config`. Note though that changes to themes here will be overwritten every time Audacity is updated.
