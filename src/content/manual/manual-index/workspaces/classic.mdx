## Classic Workspace

The Classic workspace contains these items in the toolbar:

- Play
- Stop/Pause
- Record
- Rewind to Start
- Rewind to End
- Toggle Loop Region
- Clip Gain (Envelope)
- Split Tool
- Toggle Spectral View
- Zoom in
- Zoom out
- Zoom to Selection
- Zoom to Fit Project&#x20;
- Zoom Toggle
- Cut
-

### Ruler

The Ruler is set to 'Minutes and Seconds' instead of 'Beats and Measures'&#x20;
